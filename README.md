موقع أخبار نادي الزمالك — جاهز للرفع على Netlify.

---

## هيكل الملفات

```
zamalek-news/
├── index.html        ← الصفحة الرئيسية
├── dashboard.html    ← لوحة تحكم المشرفين
├── style.css         ← ملف التنسيق
├── app.js            ← منطق JavaScript
├── assets/
│   └── favicon.svg   ← أيقونة المتصفح
└── README.md         ← هذا الملف
```

---

## التشغيل المحلي (للاختبار)

### الطريقة 1: VS Code Live Server (الأسهل)
1. افتح المجلد في VS Code
2. ثبّت إضافة **Live Server**
3. انقر بزر الماوس الأيمن على `index.html` → **Open with Live Server**

### الطريقة 2: Python
```bash
cd zamalek-news
python -m http.server 8000
# ثم افتح http://localhost:8000
```

### الطريقة 3: Node.js
```bash
npx serve .
```

---

## الرفع على Netlify

1. اذهب إلى [netlify.com](https://netlify.com) وسجّل دخولك
2. اسحب مجلد `zamalek-news` بالكامل إلى منطقة **Drag & Drop** في Netlify
3. انتظر دقيقة — سيعطيك Netlify رابطاً مباشراً للموقع
4. **انتهيت!** ✅

## التخزين

- يستخدم الموقع **localStorage** لحفظ الأخبار في متصفح المستخدم
- الأخبار تبقى محفوظة حتى بعد إغلاق المتصفح
- كل مستخدم يرى نفس الأخبار لأن التخزين يعتمد على نفس المتصفح
- لنشر موقع بقاعدة بيانات مشتركة حقيقية، يمكن لاحقاً ربطه بـ Supabase أو Firebase
